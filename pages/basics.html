Lesson Module:

Learning Objectives

After completing this lesson, you should be able to:

1. Define variables and understand their purpose in programming.

2. Differentiate between primitive and reference data types in Java.

3. Identify and describe the eight primitive data types and the String data type.

4. Declare, assign, and initialize variables in Java.

5. Understand the conventions for naming variables and assigning values to different data types.

6. Perform basic operations with variables, such as printing their values and string concatenation.

Lesson Content

1. Introduction to Variables

• A variable is a placeholder for a value.

• Variables behave as the value they contain.

• Variables allow storing different types of data, including numbers, words, sentences, and Boolean values.

2. Primitive vs. Reference Data Types

• Primitive data types: boolean, byte, short, int, long, float, double, char.

• Store data directly.

• Can hold only one value.

• Uses less memory.

• Faster.

• Reference data types (e.g., String)

• Unlimited (user defined).

• Store an address.

• Can hold multiple values.

• Uses more memory.

• Slower.

3. Primitive Data Types

• boolean: 1 bit, holds true or false:

boolean isTrue = true;
boolean isFalse = false; JAVA

• byte: 1 byte, holds integers between -128 and 127:

byte age = 25; JAVA

• short: 2 bytes, holds integers between -32,768 and 32,767:

short count = 500; JAVA

• int: 4 bytes, holds integers between -2,147,483,648 and 2,147,483,647:

int population = 1000000; JAVA

• long: 8 bytes, holds very large integers (e.g., -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807):

long distanceToMars = 54600000L; // Note the 'L' suffix JAVA

• float: 4 bytes, holds fractional numbers up to 6-7 decimal digits (must end with 'f'):

float pi = 3.14159f; JAVA

• double: 8 bytes, holds fractional numbers up to 15 decimal digits:

double gravity = 9.80665; JAVA

• char: 2 bytes, holds a single character (must be enclosed in single quotes):

char letter = 'A'; JAVA

4. Reference Data Type: String

• Reference data type.

• Stores a sequence of characters (word or sentence).

• Size varies based on the string length.

String name = "John Doe";
String message = "Hello, World!"; JAVA

5. Declaring, Assigning, and Initializing Variables.

• Declaration: specifying the data type and variable name (e.g., int x;).

• Assignment: assigning a value to a variable (e.g., x = 123;).

• Initialization: combining declaration and assignment (e.g., int x = 123;).

int x; // Declaration
x = 123; // Assignment
int y = 456; // Initialization JAVA

6. Naming Conventions and Value Assignment

• Follow camelCase for variable names (e.g., myVariable).

• Use appropriate data types for the values you want to store.

• Enclose string values in double quotes.

• Follow conventions for assigning values (e.g., floats end with 'f', longs end with 'L').

int age = 25;
double decimalOne = 22.5;
double decimalTwo = 44.3;
long population = 7900000000L;
String name = "Cottonfields"; JAVA

7. Working with Variables

• Print variable values to the console:

System.out.println(age); // Output: 25
System.out.println(name); // Output: Cottonfields JAVA

• Perform string combination using the + operator:

String greeting = "Hello, " + name; // Output: Hello, Cottonfields JAVA

• Variables behave as the value they contain and can be used in calculations and operations:

int sum = age + 10; // sum = 35
double area = decimalOne * decimalTwo; // product = 996.7 JAVA

#3: Accepting User Input in Java

Lesson Module:

Learning Objectives

After completing this lesson, you should be able to:

1. Understand the concepts of classes and objects in Java.

2. Identify the structure of a Java class and its components.

3. Explain the purpose and structure of methods in Java.

4. Recognize the importance and structure of the main method as the entry point for program execution.

5. Apply naming conventions for classes, methods, and variables in Java.

6. Describe the structure of a basic Java program, including the use of packages, access modifiers, and comments.

7. Understand the concepts of variables and data types in Java.

Lesson Content

1. Classes and Objects

• A class is a blueprint for creating objects.

• An object is an instance of a class.

• Example: A car factory (class) produces individual cars (objects).

2. Java Class Structure

• Classes are defined using the class keyword, followed by the class name and a pair of braces {}.

• The code for the class is written inside the braces.

• Example:
                        
public class MyClass {
// code goes here
} JAVA

3. Methods

• A method is a group of instructions that perform a specific task.

• Examples: adding two numbers -> printing a greeting -> getting user input.

• Methods are defined with a return type, name, parameters, and a body (code block).

• Example:
                        
public int addNumbers(int a, int b) {
return a + b;
} JAVA

4. The main Method

• The main method is a special method automatically called when a Java program is run.

• It is the starting point of execution for a Java program, and every Java program must have a main method.

• The main method serves as the entry point for the program, where the execution begins.

• Example:
                        
public static void main(String[] args) {
// program code goes here
} JAVA

5. Naming Conventions

• PascalCase: Used for naming classes (e.g., MyClas).

• camelCase: Used for naming methods and variables (e.g., myMethod, myVariable).

• snake_case: Used for naming constants (e.g., MY_CONSTANT).

• The use of different naming conventions (PascalCase for classes, camelCase for methods and variables, snake_case for constants) in Java aids code readability, maintainability, and adheres to widely accepted standards and best practices.

6. Java Program Structure

• A Java program consists of at least one class.

• The main method is defined inside a class.

• The program code is written inside the main method's body.

• Comments play an important role in documenting and explaining the code, making it easier to understand and maintain.

• Variables are used to store data in a Java program.

• Java has a variety of data types, such as int (integers), double (floating-point numbers), boolean (true/false), and String (text).

• Variables are declared with a data type and a unique name, and their values can be assigned, read, and updated throughout the program.

• Example of a Java program with each line explained:
                        
// The class declaration starts with the 'class' keyword
// The class name should be a descriptive, meaningful name
// In Java, class names are usually written in PascalCase
// The class name should also be the same name as your file name
public class Main {
                          
// The 'main' method is the entry point of the program
// The 'public' access modifier makes the method accessible from outside the class
// The 'static' keyword means the method can be called without creating an instance of the class
// The 'void' return type means the method doesn't return any value
public static void main(String[] args) {
                          
// Print a message to the console
// The 'System.out.println()' method is used to output text
System.out.println("Hello, World!");
                          
// Declare a variable to store an integer value
// Variable names should be descriptive and use camelCase
int myVariable = 42;
                          
// Print the value of the variable
System.out.println("The value of myVariable is: " + myVariable);
  }
}
